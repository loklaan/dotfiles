name: lochy:challenge
description: >-
  Self-interrogate a first draft before locking it in. A pause-and-refine pass
  that pressure-tests initial output against "could this be better, more
  powerful, more elegant — what inspirations am I missing?"
  Use after producing an initial draft, first plan, proposed examples, or any
  candidate output, before committing to it as the answer.
---

Before locking this in, challenge yourself for a second:

- How could this be better?
- Is there a cleaner framing of the problem itself that would make the pieces
  naturally cohere — ideally sharing the same underlying shape?
- Then each piece on its own — does it serve that shape, and is it
  well-crafted on its own terms?
- Are there existing inspirations in your consciousness that would amplify this?

Then revise — or justify keeping it as-is.
